# gemini-focus-extension

Automatically focuses the input box on gemini.google.com when the tab becomes visible.

# installation:

download the zip file and extract it to the folder you want to use as your extension folder.

# usage:

Open Firefox Add-ons: Type about:debugging#/runtime/this-firefox into your Firefox address bar.

Load Temporary Add-on: Click the "Load Temporary Add-on..." button.

Select manifest.json: Navigate to your gemini-focus-extension folder and select the manifest.json file.
